src/global.d.ts

Purpose

Adds a global TypeScript declaration for NodeJS.Global to optionally include a cache_manager instance.

Details

  • Imports CacheManager from src/managers.
  • Extends NodeJS.Global with cache_manager?: CacheManager.
  • Exports an empty object so the file is treated as a module.

Context

This allows attaching a shared cache manager to the Node global object while keeping TypeScript happy.
