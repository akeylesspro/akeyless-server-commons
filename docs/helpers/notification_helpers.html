src/helpers/notification_helpers.ts

Purpose

SMS and push-notification helpers for devices and mobile users.

Dependencies

  • axios, FormData, Twilio
  • Firebase messaging
  • Managers: cache_manager, logger, translation_manager
  • Firebase helpers: add_document, add_audit_record, get_nx_settings
  • Phone utilities from phone_number_helpers

SMS flow

  • ICCID → Monogoto, international → Twilio, default → Multisend.
  • send_sms expands groups, sends in parallel, and audits.
  • keep_outgoing_sms stores outbound SMS in Firestore.

Push notifications

  • push_event_to_mobile_users resolves drivers and sends FCM.
  • send_fcm_message builds and sends a multicast message.

Context

Coordinates notifications using cached settings, translations, and Firebase messaging.
