src/helpers/time_helpers.ts

Purpose

Time conversion and formatting helpers for Firebase timestamps, Dates, and string inputs.

Dependencies

  • moment, moment-timezone
  • Firebase Timestamp
  • firebase_timestamp type

Exports and behavior

  • sort_by_timestamp: comparator based on millis.
  • calculate_time_passed: returns elapsed fields and formatted strings.
  • any_datetime_to_string: converts many inputs to formatted string.
  • any_datetime_to_millis: converts many inputs to millis.
  • timestamp_to_string, timestamp_to_millis: legacy helpers.

Context

Used by services that work with Firebase timestamps and need consistent display and sorting logic.
