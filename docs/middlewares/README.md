# src/middlewares

## Contents

- `global_mw.ts`: validation and request logging middleware.
- `auth_mw.ts`: auth helpers for Firebase users and API clients.
- `error_handling.ts`: async wrapper and global error handler.
- `trim_mw.ts`: request body string trimming.
- `index.ts`: barrel exports.

## Context

Middlewares standardize request validation, authentication, logging, and error handling across services.
