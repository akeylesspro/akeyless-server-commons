src/middlewares/global_mw.ts

Purpose

Validation middleware for mandatory/optional parameters and request logging utilities.

Dependencies

  • json_failed from helpers
  • Types: MandatoryObject, MandatoryParams, LogRequests, MW
  • logger manager

Exports and behavior

  • mandatory: enforces presence, type, lengths, object keys.
  • optional: validates only when values are present.
  • request_logger: conditional request logging.

Internal logic

validateParameter checks types, lengths, array element types, and required object keys.

Context

Used to standardize input validation and optional request logging across Express routes.
