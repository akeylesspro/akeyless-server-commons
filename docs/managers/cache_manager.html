src/managers/cache_manager.ts

Purpose

Singleton in-memory cache for arrays and objects by key.

Exports and behavior

  • CacheManager.getInstance() returns a singleton.
  • setArrayData / getArrayData (defaults to empty array).
  • setObjectData / getObjectData (default optional).
  • cache_manager is the singleton instance.

Context

Used across helpers for caching settings, snapshots, and task data.
