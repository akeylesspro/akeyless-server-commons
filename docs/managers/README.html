src/managers

Contents

  • cache_manager.ts - Singleton in-memory cache manager for storing arrays and objects by key. Provides fast access to frequently accessed data populated by Firebase snapshots or Redis subscriptions
  • logger_manager.ts - Singleton logger manager providing timestamped logging with intelligent formatting, Axios-aware error handling, and table formatting for structured data
  • translation_manager.ts - Singleton translation manager providing cached translation lookups for UI messages, SMS messages, and email content with multi-language support
  • index.ts - Barrel export for managers

Context

Managers provide shared state and cross-cutting utilities used by helpers and middlewares. All managers use the singleton pattern to ensure consistent behavior across the application:

  • Cache Manager - Centralized in-memory cache for arrays and objects
  • Logger Manager - Consistent logging with timestamp formatting
  • Translation Manager - Multi-language translation lookups
